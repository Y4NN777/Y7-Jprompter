Frequently Asked Questions

What is Y7-Jprompter?

Y7-Jprompter is a free tool that converts natural language prompts into structured JSON format. This helps you create more effective prompts for AI systems by organizing your instructions clearly.

Is it really free to use?

Yes! You get 100 free conversions per day. For unlimited usage, you can add your own free Gemini API key from Google AI Studio.

Is my API key safe?

Absolutely. Your API key is stored only in your browser's local storage and is sent directly to Google's API. We never store or have access to your key on our servers.

What are complexity levels?

Complexity levels (1-7) control how detailed the JSON output will be. Level 1 gives you a simple structure, while level 7 provides the most comprehensive breakdown with all available fields.

What is Context Injection?

Context Injection (available at complexity 3+) lets you add project-specific context like your tech stack, coding style, or domain knowledge. This helps generate more tailored prompt structures.

Can I save my converted prompts?

Yes! All your conversions are automatically saved to History. You can also save favorites to your Prompt Library for quick access later.
